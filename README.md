# Django Media Storage Project

A simple Django project to **upload, store, and display media files** (images).  
This project is primarily built for **learning purposes** to understand file handling in Django.

 Features

- Upload images via a simple form
- Store uploaded images in the local `media/` folder
- Display all uploaded images in a gallery
- Handle missing or empty images safely
- Uses Django templates for rendering

 Technologies Used

- Python 3.12
- Django 5.2
- HTML (Django templates)
- SQLite (default Django database)
- Django static & media file handling
